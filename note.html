Web

Web is like a lot of connected computers which play different roles.

The Component of Web

The web consists of Internet and Server. And here is an explanation of its structure and how it works.

The internet is like the middle-man who take the requests of the user (normally the users are ordinary people like me, and the requests are like visiting a web page) and find the right server who can process the request and send the request to the server, then take the response from the server and give it back to user. The communication (including the request and response) between them follows HTTP, which stands for Hypertext Transfer Protocol and which is a standard for the communication on web.

About Programming

Computers are both stupid and smart. They can do calculation with great efficiency. However, that is the only thing they know how to do (That's why we call them computer). So we must talk to computers in a way that they could understand, which means, in programming language, the order given in which could be turned into a serie of purely computing tasks. And we shall also eliminate even the slightest mistake because computers will follow our order in any case, even when the order is catastrophic.

Avoid Repetition

When we write code, we should try our best to avoid repetition. It is far better to pack a segment of code into a method/function than copy and paste the same code everywhere you need it, even though the latter could improve the efficiency of execution of the code considering that there will be less methods/functions in the stacks of the compiler, because our code is more than something for the computer to run, it is also for people (including the code him/herself) to read, understand, maintain and improve. For example, if you find a mistake in a piece of code in a method/function, you only need to correct it inside that function, but if you do not pack it into a method/function, then you must correct it everywhere the segment of code is repeated (which means you may need to do the same correction manually thousands of times).

Computer Program

A computer program is a segment/segments of code which consist(s) of a sequence of command which a computer could understand and execute. That is the way people order the computer to do stuff. Basically, a computer without program can hardly do anything except consuming energy.

OOP (Object Oriented Programming)

OOP is a programming paradigm for development. Everything is built upon object which could be regarded as a container for variables and methods. OOP is fascinating because it provide excellent patterns for developers to write reusable and extendable code.

The basic components of OOP are class and instance. We could also define some class which extend another class, and that feature is called inheritance. The class extended is called super class and the class extending is called sub class. The sub class have everything a super class have and it could have more.

To make an analog, we could say a class is like a specie and an instance is like an individual of that specie. For example, we could say that any animal can eat, and dogs can do that too and it can also bark. And Pepper is the name of the dog of my friend. We could say animal is a class, and dog is a sub class of animal. And Pepper is an instance of dog or animal.

If we express that in code, it will be: 

class Animal():
  def eat():
    eat something......



class Dog(Animal):

  def bark():
    Wow......



pepper=Dog()

if I run pepper.bark() then Pepper would say "Wow".

The concept of element in HTML could also be regarded as class, and the elements which could be found in any HTML file are like instances of these classes. So building a web page is like creating instances of classes (including elements and the classes we create ourselves) and describing how every type of class looks like.

Class Variable

When some state for a class is independent from the situation of any particular instance, we can define a class variable, because it is not necessary (and not desirable either) to let every instance have its own copy of this value. For example, the name of the specie of dog is always dog, and it is true for the whole specie. So if we wanna add the name of the specie to dog class, we can do it like this (we write its name in upper case to imply that it is a constant variable): 

class Dog(Animal):

  NAME_OF_SPECIE="dog"

  def bark():
    Wow......

Method Overriding

We can imagine that god is a developer. And one day god wanna all animals to be vegetarian and only dogs be carnivore. So the code in animal class will be like this: 

class Animal():

  def eat():
    eat some plant......

But if god does modify the code in dog class, dogs will also be vegetarian like other animals. So in order to let dogs be carnivore, we must redefine the method eat() in dog class: 

class Dog(Animal):

  def bark():
    Wow......

  def eat():
    eat some meat......

and if I call pepper.eat(), then Pepper will go to eat some meat.

When we call some method inside a sub class, and two functions defined exactly the same way (name and input) in both super class and sub class, then the method defined in sub class, instead that one in super class, will be called.

HTML

HTML is the abbreviation of HyperText Markup Language. It's like a glue to bind everything like text, videos and images together and arrange them in a specific order. In fact, when you open a web page, what you get is some HTML code, which will then be translated by your browser to something that is displayed on the screen.So the HTML code is the orders given by the server to your browser of what to be displayed and the way to display them.

The structure of a HTML file is like this.

Concepts in HTML

Tag

Tags are used to describe how should the content be displayed. There are also tags which has no corresponding contents. These tags are called void tag

Element

Elements are components of HTML file, for example, a tag as a whole could be called an element. There are inline elements and block elements. Block elements is like an invisible box which contains the content inside it.

CSS

CSS is the abbreviation of Cascading Style Sheets. It's used to describe how should the elements in HTML file be displayed by the browser.

Build A Web Page

To build a web page, one should consider what to write and how should be displayed on screen

Structural Design

Thanks to the div tag in HTML, we can divide a page into different blocks and layers and assign different properities to them, which facilitate a lot our design. Web page designed in this way forms a tree-like structure. Tree-like structure means that you can have boxes inside boxes inside boxes etc., you can have many boxes inside one box, but it is impossible to get a box which is inside two or more boxes at the same time. Likewise, you can get branches from branches from branches etc., but every branch can only come from one corresponding mother branch.

Style Design

One can see the appearance of a web page as a bunch of invisible and/or visible boxes, which serve as the container for elements. Each box corresponds to an element. We use boxes as container because we can easily fill the screen with boxes, and boxes allows us to make adjustment without too much effort, that's why we have bricks of the shape of cuboid instead of cylinder or cone. Since a box has different layer of "package", one can assign different properties to these layers to make the web page look fancy. These boxes could be classified into different groups by giving different class names.

We could directly write the part of code for style inside the HTML file, however, to avoid repetition and increase reusability, it is better to write it in a separate CSS file and refer to it in the HTML file.

Python

Python is a programming language. Unlike HTML and CSS which can only be used for the description of web pages, Python could serve a variety of purposes, for instance, automatically generate HTML code.

Grammar

Every programming language has its own grammar. Here, the grammar means the way to express our command which could enable the computer to understand it.

One crucial part of grammar is its structure of expression. This video provide a good explanation of the rule for constructing a valid expression.

Variable

We can give names to values to let them become a variable. By allowing us to store and modify the value, variable enable us to complete more complicated tasks. More over, we can simply give variables meaningful names to increase the readability of our code. To define a variable and assign it to a value, we can write code like

a=3

in Python, which means we create a variable a and let its value to be 3 and the operator = here means "takes the value of".

Operator

There are many operators in Python, such as arithmetic operators +, -, * and /. One important point is that one operators could have different meanings in different context, for example, when + appears between two numbers, it means plus, and when it appears between two Strings, it means concatenation.

Decision And Loop

We can put a segment of code into a if branch in order to make a decision which part of code should be executed. We can also put something into a loop to repeat doing the same stuff. The following videos explain how to construct a if branch and a while loop.

We can also use a for loop to iterate the elements in a list and do something to them.

We can build code to realize a complex task by using these structure.

Function

A function is like a segment of code which could be called by its name. By defining and calling function, we can significantly avoid repetition and increase the readability of our code.

A function can take input and give output. A function ends and sends the output to where it is called when it meet the keyword return.

Here is an example of defining and calling a function:
defining a function:

def calculateSum(input1, input2):
            output=input1+input2
            return output

calling a function: 

print calculateSum(1,2)

It should print 3.

If a function does not have output, it is not necessarily a useless function, because the modification it made may still be there after it finishes. For example, the modification on an array passed in will not be canceled after the function finishes.

Module And Library

A module is a file which is used for containing Python code. The file name of a module must end with .py which indicate that it is a file for Python.

A library is a bunch of Python modules which is written by someone (could be those people who develop Python) for some purposes, for example, to realize some feature.
